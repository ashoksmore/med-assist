# MedAssist PPT Visual Style Guide

Use this style guide to keep all slides consistent and judge-friendly.

## 1) Design Principles

- Clean, clinical, and calm
- High contrast for readability
- One idea per slide
- Prefer short bullets over dense paragraphs
- Prioritize flow diagrams and before/after visuals over tables

## 2) Color Palette

Primary:
- Navy: `#0B1F3A`
- Blue: `#1E5AA8`

Support:
- Teal: `#0FA3B1`
- Slate: `#5E6C84`

Background:
- White: `#FFFFFF`
- Soft gray: `#F5F7FA`

Status accents:
- Success green: `#2E7D32`
- Caution amber: `#ED8B00`
- Error red: `#C62828`

Usage rule:
- Keep 70% white/gray, 20% navy/blue, 10% accent colors.

## 3) Typography

Recommended font family:
- Inter (or fallback: Calibri / Arial)

Sizes:
- Slide title: 38-44 pt
- Section subtitle: 24-28 pt
- Body text: 20-24 pt
- Footnotes/captions: 14-16 pt

Weight:
- Titles: Bold
- Body: Regular
- Emphasis: Semibold (avoid all-caps paragraphs)

## 4) Spacing and Layout

Margins:
- Keep at least 6-8% margin on all sides

Grid:
- Use a 12-column or simple 2-column layout
- Left-align text blocks for consistency

Bullet limits:
- Max 5 bullets per slide
- Max 10 words per bullet when possible

## 5) Slide Patterns

Title slide:
- Big title, short subtitle, team footer

Problem/Solution slides:
- Left: text
- Right: icon or simple diagram

Flow slide:
- Horizontal 5-step process with numbered circles

Metrics slide:
- 1 hero KPI card + 3 to 4 supporting metric chips

Results slide:
- Before vs After visual (two columns)

## 6) Icon and Visual Rules

- Use one icon style set only (outline or filled, not mixed)
- Avoid clipart and decorative backgrounds
- Prefer simple patient/clinician/workflow icons
- Use screenshots only if text is readable from a distance

## 7) Accessibility

- Minimum contrast ratio target: 4.5:1
- Do not rely on color alone to convey meaning
- Keep font size >= 20 pt for body text
- Avoid red/green-only comparisons without labels

## 8) Animation and Transitions

- Keep transitions minimal (Fade only)
- No complex motion effects
- Use build animations only for sequential process explanation

## 9) Recommended Slide Footer

Footer left:
- MedAssist | athenahealth Hackathon

Footer right:
- Slide number

## 10) Fast Consistency Checklist (Before Presenting)

- Titles use same style and size
- Colors follow palette rules
- Bullets are concise and parallel in tone
- Each slide has one clear message
- Demo flow and metrics wording match submission docs
